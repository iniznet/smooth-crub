# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-02-21

### Added
- Support for inline color markers (`{#color:...}`, `{#bg:...}`, `{#stroke:...}`) to customize text color, background color, and stroke color of boxes.
- New "Quest Journal" fixture demonstrating color markers.
- Updated "Chat Interface" fixture to use color markers.

### Changed
- Updated snapshot tests to reflect the new color marker rendering.
